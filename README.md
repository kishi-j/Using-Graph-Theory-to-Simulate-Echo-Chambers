# King's Certificate Project: Investigating Echo Chambers and their Formation using Graph Theory
### Two types of processes – 

•	behavioural (how people change opinions)

•	algorithmic (how networks and algorithms change connections)

### Two key measures:

•	Q(t) — Modularity: how strongly the network splits into communities 

•	r(t) — Assortativity: how similar connected users opinions are
